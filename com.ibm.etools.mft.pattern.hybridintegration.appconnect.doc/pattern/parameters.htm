Number of Events 1

When defining a custom Application to IBM App Connect, you must share a .yaml document which specifies one or more triggers. Each trigger has a name, description and wire format interface format associated with it. By specifying a number between 1 and 5, you will then get the opportunity to specify between 1 and 5 triggers, and for each of these define its name, description and format.
